README

This folders contain the replication package for 'Labor Market Institutions, Fiscal Multipliers, and Macroeconomic Volatility' published in the Journal of Applied Econometrics by Maximilian Boeck, Jesús Crespo Cuaresma, and Christian Glocker. Codes are written in R and Matlab.

To replicate the results of the paper, please follow these instructions. First, download and unzip the folderboth the /scripts and /data folder into a directory. Second, create a new directory /results. Third, set your working directory in R and Matlab to the created folder containing /scripts, /results, and /data. Fourth, execute the R and Matlab scripts in the given order. This creates all figures of the paper.

In the /data folder we provide another README with a documentation on the data sources, which are also given in the appendix of the paper.

Correspondingly, we will maintain a GitHub repository containing the replication files: https://github.com/mboeck11/employment-multiplier.
