README

This folders contain the replication package for 'Labor Market Institutions, Fiscal Multipliers, and Macroeconomic Volatility' published in the Journal of Applied Econometrics by Maximilian Boeck, Jesús Crespo Cuaresma, and Christian Glocker. Codes are written in R and Matlab.

To replicate the results of the paper, please follow these instructions. First, unzip each folder and put them into the same directory. In order that everything runs smoothly, please unzip /data, /dsge, and /empirics in the same folder. To replicate all figures of the paper, please execute the following scripts: i) for the theoretical analysis, execute dsge/main.m. This is done in Matlab and the Dynare toolbox. We provide further information on the version in a README in the folder; ii) for the empirical analysis, execute empirics/main.R. This is done in R. This creates all figures of the paper. Please note that the execuation can take considerable computing time.

In the /data folder we provide another README with a documentation on the data sources, which are also given in the appendix of the paper.

Correspondingly, we will maintain a GitHub repository containing the replication files: https://github.com/mboeck11/replication-employment-multiplier.

If you have any further questions, please do not hesitate to contact us. Mail: maximilian.boeck@wu.ac.at, jesus.crespo.cuaresma@wu.ac.at, christian.glocker@wifo.ac.at.
